FAQ

What is a text game scaffold?

It's a carefully written prompt (sometimes several files) that sets up an AI chatbot to act as a game master. It defines the world, the rules, and the tone so the game stays consistent and fun instead of wandering off course.

Is this like a TTRPG or D&D?

Close cousin. These play like a one-player tabletop RPG where the AI is the game master — you have stats, make choices, face consequences, and can attempt anything your character plausibly could. Crossed with classic interactive fiction: it's all told in prose, turn by turn, in a chat window. No dice, no rulebook, no scheduling four friends.

Can I play these with ChatGPT, Gemini, or Grok?

Mostly, with some tweaking. The games are written and tested as Claude skills. Other AI chatbots can run them too, but they have different prompt structures and length limits — a long multi-file game may need to be condensed or restructured to fit. The single-file games port most easily. If you adapt one and it plays well, that's a modified version, and your mileage may vary.

What's a Claude skill?

A Claude skill is a folder containing a main instruction file (and sometimes supporting reference files) that Claude loads automatically. If you use Claude, downloading the skill is the smoothest way to play. If not, copy-paste works great. The .skill download is just a ZIP archive with a different extension — you can open it with any unzip tool.

What do "Well-tested" and "Beta" mean?

Well-tested games have been through many full play sessions and their mechanics are smoothed out. Beta games are complete and playable, but newer — they've had fewer run-throughs, so you might hit a rough edge. Beta games graduate once they've earned it.

Is this free?

Yes — all the games here are free to download, copy, remix, and share.
