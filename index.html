FAQ

What is a text game scaffold?

It's a carefully written prompt (sometimes several files) that sets up an AI chatbot to act as a game master. It defines the world, the rules, and the tone so the game stays consistent and fun instead of wandering off course.

Do I need a specific AI to play?

No. The scaffolds are written as Claude skills, but the full text of every game is available on its page so you can paste it into ChatGPT, Grok, Gemini, or any other chatbot that accepts long prompts.

What's a Claude skill?

A Claude skill is a folder containing a main instruction file (and sometimes supporting reference files) that Claude loads automatically. If you use Claude, downloading the skill is the smoothest way to play. If not, copy-paste works great.

Is this free?

Yes — all the games here are free to download, copy, remix, and share.
